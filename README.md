# Nginx Log Analyzer

## About

This project is a simple Bash shell script that analyzes an Nginx access log file.

The script displays:

- Top 5 IP addresses
- Top 5 requested paths
- Top 5 status codes
- Top 5 user agents

## Technologies Used

- Bash
- Linux
- AWK
- Sort
- Uniq
- Head

## Files

```
nginx-log-analyzer/
```

## Run

Make the script executable.

```bash
chmod +x log-analyzer.sh
```

Run the script.

```bash
./log-analyzer.sh
```
